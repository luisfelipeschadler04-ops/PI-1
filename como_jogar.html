Como Jogar - instruções completas

OBJETIVO:

Você está preso em um corredor com 4 salas. Cada sala representa um tipo de apego da Teoria de Bowlby. Para escapar, resolva todos os desafios de cada sala.

PASSO A PASSO:

  • Use as setas do teclado para mover seu personagem
  • Aproxime-se de objetos brilhantes e pressione ESPAÇO para coletar pistas
  • Colete as 3 pistas da sala para desbloquear os puzzles
  • Cada sala tem 3 puzzles — resolva todos para abrir a porta
  • Erro = −1 vida | Sem vidas = reinicia a sala
