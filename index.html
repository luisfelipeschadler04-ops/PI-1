Como Jogar — Instruções Completas

Objetivo:

Você está preso em um corredor com 4 salas. Cada sala representa um tipo de apego da Teoria de Bowlby. Para escapar, resolva todos os desafios de cada sala.

Passo a Passo:

  • Clique nos objetos brilhantes ✨ para coletar pistas
  • Colete as 3 pistas da sala para desbloquear os puzzles
  • Clique na porta 🔒 para iniciar os puzzles
  • Cada sala tem 3 puzzles — resolva todos para abrir a porta
  • Resposta errada = −1 vida ❤ | Sem vidas = reinicia a sala
  • Complete todas as 4 salas para escapar do corredor!

Dica:

Leia todas as pistas com atenção antes de responder — elas contêm as informações necessárias para resolver os puzzles.
